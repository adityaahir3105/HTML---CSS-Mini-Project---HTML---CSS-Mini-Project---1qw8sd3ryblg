Smart Parking

Project

Introduction

This project is to make manless parking area which can operated by it self. Here we are aded three features in it to make it possible to operate without any interface of man. In this we will display the vacant place in parking area by priority of vacant places. Than will check whether the parking is properly done or or not. Based on that the exit of the area will we opened or closed for driver. This how a small idea of manless parking is implemented throught this module.

Working:

Here, we are using a 8*3 priority encoder which will take output of ir object detector module as its input. For that we have to put 8 ir sensor at 8 parking spots. The output of the ir sensor will be feeded in input of 8*3 encoder . As its inputs are active low the output of the ir will be perfect input for it. Than the output of the encoder will be bcd number of highest priority. The output of the encoder is active low so we need to implement not gate to all outputs. This output will be feeded to the 7447 driver ic of ssd. Ssd will display acroding to the input. This how the vacant place will be displayed in the ssd. After that initially the 0 will be saved in the register ic 74194. When any vehical will be entered in parking the number displayed on ssd will be allocated to that. Now after proper parking the output of the encoder changes and that will be compared with saved number in register by ic 7485. If the both compared numbers are not same than we can conclude that parking is correct. So the exit gate will be opened for that driver. And if the compared numbers are same it means that parking is not done properly. So the exit gate will not open. After every succesfull parking clock will be given to 74194 ic and the number displaying on the ssd will be saved in register . And again next time it will be compared with the output of encoder after parking and results will be taken. So, this how it will work without any human interface.
